## modify_attr     -> 修改属性  
#### 参量（param）：

构成方式：属性名 + 修改方向 + 数值 + 百分号（可选） + 比例基准（有百分号时可选，空表示b） + (基准属性)
*属性名*  

    ATK = 攻击  
    HP = 当前血量  
    MHP = 生命上限  
    SPD = SPEED = 速度  
    NRG = ENERGY = 能量  

*修改方向*  

    \+  增加  
    \-  减少  
    =   赋值/设置/修改为  

*数值*  

    不为0的正整数（暂定）

*百分号（%）*

    %   数值是否为百分比  

*比例基准（bmr）*

    b   基础数值（角色本体数值）
    m   最大数值（角色的数值上限（常见于生命、能量等有明确最大值的属性）
    r   当前属性（角色当前数值）

*基准属性*

    (attr)  基准属性名（无该项意味以被修改属性为基准）

**e.g.**

**ATK+10%b**    -> 提高等同于10%基础攻击力的攻击  
**HP+20**       -> 恢复20生命值（无法超过生命上限）   
**MHP-1%r**     -> 最大生命值减少1%（由于最大生命值在局内不随受伤改变，因此b/r/m是一个效果）  
**ENERGY+50%m** -> 恢复50%最大能量（等效于 *NRG+50%m* ）  
**SPD=10**      -> 设置速度为10  
**ATK=200%r**   -> 攻击力翻倍（等效于 *ATK+100%r* ）   
**ATK+50%r(SPD)**   -> 提高等同于50%速度值的攻击  
**HP+300%r(NRG)**   -> 恢复等同于300%当前能量的生命值  

#### 模式（mode）

| 关键字               | 含义        |
| ----------------- | --------- |
| `self`            | 自己        |
| `target`          | 触发事件里默认目标 |
| `source`          | 伤害来源      |
| `randomEnemy:n`   | 随机 n 个敌人  |
| `allAllies`       | 全场友军      |
| `lowestHpEnemy`   | 血量最低敌人    |
| `highestAtkEnemy` | 攻击最高敌人    |
| `exceptSelf`      | 排除自己      |
| `filterRace:orc`  | 再过滤兽人     |


**全量使用**

    {
        "type": "modify_attr",
        "param": "ATK+10",
        "mode": "self"
    }


### cause_damage    -> 造成伤害

#### 参量（param）

构成方式：数值 + 百分号 + 比例基准 + 属性名

*数值*

 不为0的正整数（暂定）

 *百分号（%）*

    %   数值是否为百分比  

*比例基准（bmr）*

    b   基础数值（角色本体数值）
    m   最大数值（角色的数值上限（常见于生命、能量等有明确最大值的属性）
    r   当前属性（角色当前数值）

*属性名*  

    ATK = 攻击  
    HP = 当前血量  
    MHP = 生命上限  
    SPD = SPEED = 速度  
    NRG = ENERGY = 能量  

**e.g.**

**80%rATK**     -> 造成80%当前攻击的伤害
**20%mMHP**     -> 造成20%最大生命值的伤害  
**5**           -> 造成5点固定伤害
