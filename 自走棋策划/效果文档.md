效果格式构成

属性关键词 + 修改方向 + 数值 + 是否百分比 + 百分比基准（opt）

属性关键词：
SPD -> speed
HP -> hp
ATK -> atk
MHP -> max hp
NRG -> energy

修改方向：
\+ -> 增加
\- -> 减少

数值：
正整数（暂定）

是否百分比：
% -> 若有该符号则表示数值为百分数
- 百分比基准：
- b -> 基础值百分比
- m -> 最大值百分比
- r -> 当前值百分比

e.g.
1. SPD+10%b -> 增加基础速度值10%的速度
2. HP-10 -> 受到10点伤害
3. HP+10%m -> 恢复最大生命值10%的生命
4. NGR+1 -> 恢复1点能量
5. ATK-10%b -> 减少10%基础攻击力的攻击
